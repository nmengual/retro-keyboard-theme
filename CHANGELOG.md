# Changelog

## 1.1.0

- Improved color contrast for WCAG compliance across both themes
- Added 30+ new UI tokens: inlay hints, sticky scroll, command center, debug toolbar, merge conflicts, overview ruler, settings, welcome page, charts, testing
- Added language-specific syntax highlighting for Python, Rust, Go, Shell, YAML, TOML, SQL, Dockerfile, and GraphQL
- Extended semantic token colors for better language server support
- Reduced red overuse — cursor, active borders, and `this/self` now use distinct warm tones
- Increased diff editor visibility
- Improved light theme color depth for readability on warm backgrounds

## 1.0.0

- Initial release with dark and light variants
- Warm, retro color palette inspired by mechanical keyboard pastel keycaps
- Full syntax highlighting for JavaScript, TypeScript, HTML, CSS, JSON, Markdown, and PHP
