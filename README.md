# Retro Keyboard Theme

A warm, retro color theme for VS Code inspired by colorful mechanical keyboards with pastel keycaps.

## Variants

### Retro Keyboard Dark

A cozy dark theme with warm brown tones and vibrant pastel accents.

### Retro Keyboard Light

A soft light theme with cream backgrounds and rich, readable colors.

## Color Palette

| Role | Dark | Light |
|------|------|-------|
| **Keywords** | `#E06060` | `#C04040` |
| **Functions** | `#8AAEC4` | `#2E6E8A` |
| **Strings** | `#8BBF9C` | `#3E8A5E` |
| **Types/Classes** | `#E8C060` | `#8A6A10` |
| **Variables** | `#A8C8DC` | `#3A6A7A` |
| **Parameters** | `#D4854A` | `#A86828` |
| **Comments** | `#8A7E6E` | `#7A6E5E` |
| **Decorators** | `#B89ED6` | `#7E5EAA` |
| **Interfaces** | `#88CCC0` | `#3A8A80` |

## Accessibility

- WCAG contrast-compliant colors across both themes
- No reliance on color alone for semantic meaning
- Carefully tuned light theme with deepened colors for readability

## Installation

### From the Marketplace

Search for **"Retro Keyboard Theme"** in the VS Code Extensions panel.

### Manual

```bash
git clone https://github.com/nmengual/retro-keyboard-theme.git ~/.vscode/extensions/retro-keyboard-theme
```

Then reload VS Code and select the theme from **Preferences > Color Theme**.
